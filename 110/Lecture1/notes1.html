Intro to HTML

Hypertext Markup Language
- the primary language used to structure and format content of web pages on the internet.

-gives a set pf predefined tags pr elements that define the structure, layout, and presentation of the info displayed on a web page.

-its considered the standard bc its widely adopted and supported by all major browsers.

-consistent / uniform

Web Pages

Documents viewed through a web browser.
-HTML is specifically made for creating web pages

-can contain various types of content (text, images, videos. forms, and interactive elements).

-HTML changes how the content is organized and displayed on a web page

-HyperText-

The method by which internet users navigate web.

-hyperlinks (leads to new pages)

-hyper=not linear

-refers to building connections between different HTML pages.

Markup Language

A type of computer language used to annotate or “mark up” text with formatting and structuring.

-is what HTML tags do to the text inside of them

-bold or italicized

-tags define structure and meaning of content (headings, paragraphs, lists, links, images, and other elements)

Tags

The words between angle brackets ("< >") which allow graphics, images, and tables to be viewed on webpage.

<-tagname->content
  • header : introductory of page or section

  • nav : navigation links.

  • article: self-contained piece of content.

  • section: defines thematic grouping of content.

  • aside:content tangentially related to main content.

  • footer: footer of document or section.

void element- content model never allows it to have content but can have attributes

Page Structure

!DOCTYPE: defines document type to be HTML

html: encloses complete HTML document

head:documents header

title:document title

body:documents body / visible page content

h1: describes heading

p:paragraph

-self closing tags(meta dont require closing tag)

-paired tags (title requires)

-formatting(use consistent spacing and case)





Links and Images

Links point to a different HTML document.

Images pull another resource into the page.

a - anchor tag / defines hyperlink <-/a>

href=””- Hyper text reference , an attribute (tell more about tag) which indicates the link’s destination.

A website is just a collection of HTML files organized into folders. To refer to those files from inside another file URLs (uniform resource locator) are used.

Types of Links
relative-points to another file in your website

root relative- to root of entire website, start with forward slash, wont work without an actual server.

absolute - most detailed way to refer a web resource

target - specifies where to open linked document.

_self - default. opens the document in the same window/tab as it was clicked.

_blank - tells browser to open the link in a new window.

_parent - opens document in the parent frame.

_top - opens document in full body window.


Images
img: used to embed image in HTML page , creates a holding space for referenced image

- has two required attributes

src- specifies the path (URL) to image

alt- specifies alternate text for image, if image cannot be displayed

- size of the images can be changed using height and width attributes.

-setting only one of them will cause image to scale proportionally, stretching the image.
